# Student Management System

## Project Description
The Student Management System is a Java-based console application that helps manage student records. 
Users can add students, view student details, and search for students by their ID or name.

## Features
- Add student records
- Display all students
- Search student by ID
- Search student by name
- Store multiple students using ArrayList
- Simple and user-friendly console output

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList (Java Collections Framework)

## Project Structure
src/
├── Main.java
├── Student.java
└── StudentManager.java

## How to Run

1. Clone the repository:
   git clone <repository-url>

2. Open the project in any Java IDE such as:
   - IntelliJ IDEA
   - Eclipse
   - VS Code

3. Compile the Java files:
   javac src/*.java

4. Run the program:
   java Main
